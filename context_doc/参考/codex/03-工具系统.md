# Codex - 工具系统

## 工具类型
- **Function Tools**: JSON Schema 定义的可调用函数
- **Freeform Tools**: 自然语言描述 + handler
- **MCP Tools**: 来自远程 MCP 服务器
- **Discoverable Tools**: 运行时发现的工具
- **Dynamic Tools**: 客户端提供的工具规格

## 工具 Handlers (tools/handlers/)
每个工具有独立的 handler 文件：
- `shell.rs` - 执行 Shell 命令
- `read_file.rs` - 读取文件
- `apply_patch.rs` - 应用 Git 补丁
- `js_repl.rs` - 执行 JavaScript
- `mcp.rs` - 调用 MCP 服务器工具
- `multi_agents.rs` - 生成子 Agent
- `request_permissions.rs` - 请求用户权限
- `request_user_input.rs` - 请求自由文本输入
- 等等 (20+ handlers)

## 工具编排器 (tools/orchestrator.rs)

执行管线：
```
1. 审批阶段：检查是否需要审批 → 获取用户决定
2. 沙箱选择：根据策略和审批状态选择沙箱级别
3. 尝试执行：在选定沙箱中运行工具
4. 网络审批：处理网络策略决定
5. 重试逻辑：拒绝时升级沙箱，重新执行
```

## 审批与沙箱策略

### 三步审批
1. **审批需求检查**: Skip / Forbidden / NeedsApproval
2. **请求审批**: `ExecApprovalRequestEvent` → 等待 `ReviewDecision`
3. **执行或拒绝**

### 沙箱策略
- **FileSystemSandboxPolicy**: 只读 / 限定路径写入 / 无限制
- **NetworkSandboxPolicy**: 阻止 / 允许特定域名
- **WindowsSandboxLevel**: None / AppContainer / HyperV
