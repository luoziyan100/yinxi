# OpenAI Codex CLI - 项目概览

## 基本信息
- **来源**: OpenAI 官方
- **语言**: Rust (核心) + Node.js (CLI 入口)
- **定位**: 终端 AI 编程 Agent
- **架构特点**: Node.js 薄壳 → Rust 二进制，高性能异步架构

## 目录结构
```
codex/
├── codex-cli/                    # Node.js CLI 包装器
│   ├── bin/codex.js              # 入口：检测平台，spawn Rust 二进制
│   └── vendor/                   # 各平台预编译的 Rust 二进制
│
├── codex-rs/                     # Rust 核心实现 (356+ 源文件)
│   └── core/src/
│       ├── codex.rs              # 主 Session 状态机 (7400+ 行)
│       ├── client.rs             # OpenAI API 客户端
│       ├── client_common.rs      # Prompt/ResponseEvent 类型
│       ├── terminal.rs           # 终端检测
│       ├── agent/                # 多 Agent 生成与控制
│       ├── config/               # TOML 配置管理
│       ├── context_manager/      # 对话历史与压缩
│       ├── tasks/                # 任务系统 (Regular, Review, Compact, Undo...)
│       ├── tools/                # 工具系统 (handlers, router, orchestrator)
│       └── ...                   # auth, MCP, sandbox, permissions 等 60+ 模块
│
├── shell-tool-mcp/               # MCP shell 工具服务器
├── sdk/                          # SDK
└── scripts/                      # 构建脚本
```

## 核心设计理念
1. **Turn-Based 状态机**: 一切围绕"轮次"组织，每轮是一次 request-response 循环
2. **Streaming-First**: 无缓冲，API 事件直接流式传输到客户端
3. **Policy-Based Sandbox**: 执行沙箱不硬编码，由策略驱动
4. **多 Agent**: 通过创建新 Session 实例实现 Agent 分裂
