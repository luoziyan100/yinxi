# Pi - 工具系统与扩展

## 内置工具

每个工具遵循相同模式：
```typescript
const tool: AgentTool = {
  name: "read",
  description: "Read file contents",
  parameters: Type.Object({
    path: Type.String({ description: "File path" }),
    line_start: Type.Optional(Type.Number()),
    line_end: Type.Optional(Type.Number()),
  }),
  execute: async (input, context) => {
    return { success: true, output: "..." }
  }
}
```

### 7 个内置工具
| 工具 | 用途 |
|------|------|
| `bash` | Shell 命令执行（流式+截断） |
| `read` | 文件读取（支持行范围） |
| `write` | 文件创建/覆盖 |
| `edit` | 精确文本替换（查找→替换） |
| `grep` | 正则搜索（尊重 .gitignore） |
| `find` | Glob 文件发现 |
| `ls` | 目录列表 |

## Bash 执行器 (bash-executor.ts)

关键特性：
- **流式输出**: `onChunk` 回调实时传输输出
- **取消支持**: `AbortSignal` → `killProcessTree()`
- **ANSI 清理**: 自动剥离终端控制码
- **截断**: 超过阈值时写入临时文件
- **二进制安全**: 检测并替换二进制输出

```typescript
executeBash(command, options) {
  const child = spawn(shell, ['-c', command], { detached: true })

  child.stdout.on('data', (data) => {
    const text = sanitizeBinaryOutput(stripAnsi(data))
    options.onChunk?.(text)       // 流式回调

    if (totalBytes > MAX_BYTES && !tempFile) {
      tempFile = createWriteStream(getTempPath())  // 大输出写文件
    }
  })

  options.signal?.addEventListener('abort', () => {
    killProcessTree(child.pid)    // 杀掉整个进程树
  })
}
```

## 扩展系统 (extensions/)

扩展是 TypeScript 模块，可 hook 进：
- 工具执行（包装/修改/替换）
- Agent 事件（agent_start, agent_end, turn_start 等）
- UI 自定义（widgets, overlays）
- 快捷键（自定义快捷键）
- 命令（自定义斜杠命令）

### 扩展接口
```typescript
export default function(api: ExtensionAPI) {
  // 注册自定义工具
  api.registerTool({
    name: "deploy",
    description: "...",
    parameters: {...},
    execute: async (input, context) => {...}
  })

  // 注册自定义命令
  api.registerCommand("stats", { ... })

  // 监听事件（可拦截/修改）
  api.on("tool_call", async (event, ctx) => {
    event.args.command = sanitize(event.args.command)
  })

  // 提供 UI
  api.on("agent_start", async (event, ctx) => {
    ctx.ui.setWidget("my-widget", { lines: ["Custom line"] })
  })
}
```

### 扩展发现路径
```
~/.pi/agent/extensions/
.pi/extensions/             (项目级)
~/.agents/extensions/       (Agent Skills 标准)
CLI --extension flags
settings.json packages 数组
```

### 工具包装（Extension Hooks）
```
原始工具 → wrapToolWithExtensions(tool, runner)
    ↓
before: 发出 tool_call 事件
    ├─ 扩展可修改参数
    ├─ 扩展可跳过执行
    └─ 扩展可提供自定义结果
    ↓
执行原始工具（如果未跳过）
    ↓
after: 发出 tool_result 事件
    └─ 扩展可修改结果
```

## 资源发现系统 (resource-loader.ts)

统一加载：
- **Extensions**: `.ts/.js` 模块
- **Skills**: SKILL.md frontmatter 格式
- **Prompt Templates**: Markdown 文件
- **Themes**: JSON 颜色/样式文件
- **Context Files**: AGENTS.md / CLAUDE.md / SYSTEM.md

上下文文件搜索路径（从 cwd 向上遍历）：
```
~/.pi/agent/AGENTS.md
父目录 → cwd: AGENTS.md / CLAUDE.md
```
