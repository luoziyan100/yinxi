# Pi Coding Agent - 项目概览

## 基本信息
- **来源**: 开源社区 (pi-mono monorepo)
- **语言**: TypeScript
- **定位**: 终端 AI 编程 Agent，支持扩展系统
- **架构特点**: 事件驱动、树形会话、扩展插件系统、多模式（交互/打印/RPC）

## Monorepo 结构
```
pi-mono/
├── packages/
│   ├── coding-agent/         # 主编程 Agent CLI & SDK
│   │   └── src/
│   │       ├── cli.ts        # CLI 入口和参数解析
│   │       ├── main.ts       # 主执行逻辑（初始化编排）
│   │       ├── index.ts      # SDK 导出
│   │       ├── cli/          # CLI 工具（参数、配置选择等）
│   │       ├── core/         # 核心业务逻辑
│   │       │   ├── agent-session.ts     # 核心：会话状态与事件循环
│   │       │   ├── system-prompt.ts     # 系统提示构建
│   │       │   ├── session-manager.ts   # JSONL 持久化
│   │       │   ├── settings-manager.ts  # 用户设置
│   │       │   ├── model-registry.ts    # 模型发现
│   │       │   ├── bash-executor.ts     # 命令执行（流式+取消）
│   │       │   ├── event-bus.ts         # 事件系统
│   │       │   ├── resource-loader.ts   # 扩展/技能/主题加载
│   │       │   ├── skills.ts            # 技能定义
│   │       │   ├── tools/               # 内置工具 (bash,read,write,edit,grep,find,ls)
│   │       │   ├── extensions/          # 扩展系统 (loader,runner,wrapper)
│   │       │   ├── compaction/          # 上下文压缩
│   │       │   └── export-html/         # HTML 导出
│   │       ├── modes/        # I/O 模式
│   │       │   ├── interactive/  # 交互式 TUI
│   │       │   ├── print-mode.ts # 单次执行模式
│   │       │   └── rpc/          # JSON-RPC 嵌入模式
│   │       └── utils/        # 工具函数
│   ├── agent/                # pi-agent-core (Agent 框架)
│   ├── ai/                   # pi-ai (LLM 工具包)
│   └── tui/                  # pi-tui (终端 UI 库)
└── scripts/
```

## 核心设计理念
1. **事件驱动核心**: 一切通过 Agent 事件流动
2. **异步事件队列**: 事件串行处理，防止竞态
3. **树形会话**: 会话是 DAG，不是线性历史
4. **扩展系统**: 扩展可 hook 进每个主要操作
5. **三种模式**: 交互式 TUI / 单次打印 / JSON-RPC 嵌入
