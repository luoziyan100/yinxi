# Pi - 会话持久化与上下文管理

## 树形会话结构

会话不是线性的，而是 DAG（有向无环图）：
```
消息 A
  ├─ 消息 B (工具调用)
  │  ├─ 消息 C (响应)
  │  │  └─ 消息 D (用户继续)
  │  └─ 消息 E (替代分支)
  │     └─ 消息 F
  └─ 消息 G (另一个分支)
```

每个条目有 `id/parentId`，支持：
- 树导航（跳转到任意点）
- 分支（从中间 fork）
- 标签书签
- 分支摘要

## SessionManager

```typescript
class SessionManager {
  // 文件操作
  static create(cwd, sessionDir?)        // 新建会话
  static open(path, sessionDir?)         // 打开已有
  static continueRecent(cwd)             // 继续最近的

  // 消息管理
  appendMessage(message)                 // 添加到当前分支
  appendCompactionEntry(...)             // 添加压缩摘要

  // 上下文构建
  buildSessionContext(): SessionContext  // 获取 LLM 消息

  // 树操作
  getTree(): SessionTreeNode[]           // 完整树结构
  getBranch(): SessionEntry[]            // 到根的线性历史
  fork(entryId)                          // 从某点创建分支
  navigateTree(targetId)                 // 切换分支
}
```

## 消息类型

### 标准 LLM 消息
- `UserMessage` / `AssistantMessage` / `ToolResultMessage`

### Pi 扩展消息
| 类型 | 用途 |
|------|------|
| `BashExecutionMessage` | 命令执行记录（output, exitCode, truncated） |
| `CustomMessage` | 扩展自定义消息（display 控制 UI 可见性） |
| `BranchSummaryMessage` | 分支摘要 |
| `CompactionSummaryMessage` | 压缩摘要 |

## 上下文压缩 (compaction/)

### 触发条件
- **溢出**: 上下文超过硬限制 → 压缩 + 重试
- **主动**: 接近限制 → 压缩但不重试

### 压缩流程
```typescript
compact(options) {
  // 1. 确定保留内容（最近的消息）
  const keepFromIndex = findSplitPoint(messages, keepRecentTokens)

  // 2. 提取要总结的旧消息
  const toSummarize = messages.slice(0, keepFromIndex)

  // 3. 调用 LLM 生成摘要
  const summary = await completeSimple({
    model,
    systemPrompt: SUMMARIZATION_SYSTEM_PROMPT,
    messages: [{ role: 'user', content: serializeConversation(toSummarize) }]
  })

  // 4. 保存压缩条目
  sessionManager.appendCompactionEntry({ summary, firstKeptEntryId, tokensBefore })
}
```

## 错误恢复

### 自动重试
- 可重试错误：overload, rate limit, 500, 503
- 指数退避：2s → 4s → 8s（最大 60s）
- 最大重试次数：3

### 溢出恢复
```typescript
if (isContextOverflow(error) && !_overflowRecoveryAttempted) {
  await compact()
  await agent.prompt(lastUserMessage)  // 压缩后重试
  _overflowRecoveryAttempted = true
}
```
