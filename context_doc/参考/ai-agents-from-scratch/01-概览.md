# AI Agents From Scratch - 项目概览

## 基本信息
- **来源**: 教学项目（英文版）
- **语言**: JavaScript (Node.js ESM)
- **定位**: 从零构建 AI Agent + 仿 LangChain/LangGraph 框架
- **核心理念**: "先理解原理再用框架"
- **LLM**: node-llama-cpp (本地推理) + OpenAI API

## 目录结构
```
ai-agents-from-scratch/
├── examples/                    # 第一阶段：10 个渐进示例
│   ├── 01_intro/               # 基础 LLM 推理
```

## 两阶段学习路径
- **阶段 1 (examples/)**: 从基础 LLM 调用 → 成熟的 ReAct Agent
- **阶段 2 (src/ + tutorial/)**: 理解并实现 LangChain/LangGraph 核心概念
