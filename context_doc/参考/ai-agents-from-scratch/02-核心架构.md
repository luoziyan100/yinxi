# AI Agents From Scratch - 核心架构模式

## 1. Runnable：万物之基 (core/runnable.js)

所有组件（Prompt、LLM、Parser、Chain、Agent）都实现 Runnable 接口：

```javascript
class Runnable {
  async invoke(input, config)      // 单次调用
  async *stream(input, config)     // 流式输出
  async batch(inputs, config)      // 并行批量（Promise.all）
  pipe(other) → RunnableSequence   // 组合：runnable1.pipe(runnable2)
  _call(input, config)             // 子类实现点
}

class RunnableSequence {
  // 链式执行：output₁ → input₂ → output₂ → ...
}
```

**为什么重要**：
- 可组合：`prompt.pipe(llm).pipe(parser)`
- 统一执行模型
- 允许中间件（callbacks, logging, 错误处理）

## 2. 消息系统 (core/message.js)

```javascript
BaseMessage (abstract)
├── SystemMessage       (role: system)     // LLM 指令
├── HumanMessage        (role: user)       // 用户输入
├── AIMessage           (role: assistant)  // 模型输出（可含 toolCalls）
└── ToolMessage         (role: tool)       // 工具执行结果
```

消息流：
```
SystemMessage("You are...")
  → HumanMessage("What time?")
  → AIMessage(content: "", toolCalls: [{name: "get_time"}])
  → ToolMessage(content: "14:30", toolCallId: "...")
  → AIMessage("It's 2:30 PM")
```

## 5. ReAct Agent 模式 (examples/09_react-agent/)

三步循环：
```
1. Thought: LLM 推理下一步
2. Action: LLM 调用工具
3. Observation: 工具结果反馈给 LLM
... 重复直到 Answer
```

系统提示结构：
```
CRITICAL: Follow this EXACT pattern:
Thought: [Explain calculation needed]
Action: [Call ONE tool]
Observation: [Wait for result]
... (repeat)
Answer: [Final answer - STOP HERE]
```

Agent 循环：
```javascript
async function reactAgent(userPrompt, maxIterations = 10) {
  let iteration = 0
  while (iteration < maxIterations) {
    const response = await session.prompt(prompt, { functions })
    if (response.includes("Answer:")) return response
    iteration++
  }
}
```

## 7. 回调管理器 (utils/callback-manager.js)

class CallbackManager {

## 关键设计模式

| 模式 | 用法 |
|------|------|
| Template Method | `invoke()` 调用 `_call()` |
| Composition | `pipe()` 创建 RunnableSequence |
| Strategy | 运行时配置覆盖 |
| Decorator | Callbacks 包装执行 |
| Generator | `async*` 流式生成器 |
| Registry | 消息类型、工具、记忆系统 |
| Lazy Init | 模型首次使用时加载 |
| Factory | `ChatPromptTemplate.fromMessages()` |
