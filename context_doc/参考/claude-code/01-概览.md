# Claude Code - 项目概览

## 基本信息
- **来源**: Anthropic 官方
- **语言**: TypeScript/Node.js (核心二进制单独分发)
- **定位**: 终端 AI 编程 Agent
- **注意**: 此仓库是**插件和文档仓库**，核心运行时源码不在此处

## 架构特点
- 基于**插件系统**扩展
- 渐进式信息加载 (Progressive Disclosure)
- 命令/Agent/技能/Hooks/MCP 五大子系统

## 仓库结构
```
claude-code/
├── .claude-plugin/
│   └── marketplace.json         # 插件市场注册表
├── plugins/                     # 13 个官方插件
│   ├── agent-sdk-dev/          # Agent SDK 开发工具
│   ├── claude-opus-4-5-migration/ # 模型迁移
│   ├── code-review/            # 自动 PR 代码审查
│   ├── commit-commands/        # Git 工作流自动化
│   ├── explanatory-output-style/ # 教学模式
│   ├── feature-dev/            # 7 阶段功能开发 ★
│   ├── frontend-design/        # 前端设计辅助
│   ├── hookify/                # 自定义 Hook 创建
│   ├── learning-output-style/  # 交互学习模式
│   ├── plugin-dev/             # 插件开发工具包 ★
│   ├── pr-review-toolkit/      # PR 审查专业化
│   ├── ralph-wiggum/           # 迭代 AI 循环
│   └── security-guidance/      # 安全漏洞警告
├── examples/                    # 配置示例
│   └── settings/               # 权限和沙箱设置
└── scripts/                     # 仓库管理脚本
```

## 插件标准结构
```
plugin-name/
├── .claude-plugin/
│   └── plugin.json              # 元数据（名称、描述、版本）
├── commands/                    # 斜杠命令（Markdown 文件）
├── agents/                      # 专业化 Agent 定义（Markdown）
├── skills/                      # 可复用知识包
│   └── skill-name/
│       ├── SKILL.md            # 核心技能定义（必须）
│       ├── references/         # 参考文档
│       ├── scripts/            # 可执行代码
│       └── assets/             # 输出模板
├── hooks/                       # 事件驱动自动化
│   ├── hooks.json              # Hook 配置
│   └── handler.py/.sh          # Hook 实现
└── .mcp.json                   # MCP 服务器配置
```
