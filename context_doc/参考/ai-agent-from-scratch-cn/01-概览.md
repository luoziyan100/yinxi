# AI Agent From Scratch 中文版 - 项目概览

## 基本信息
- **来源**: 教学项目（中文版，与英文版 ai-agents-from-scratch 配套）
- **语言**: JavaScript (Node.js ESM)
- **定位**: 从零构建 AI Agent + 仿 LangChain/LangGraph 框架（中文文档）
- **依赖**: dotenv, node-llama-cpp, openai

## 与英文版的关系
结构和代码基本一致，区别在于：
- 文档和注释为中文
- CONCEPT.md / CODE.md 为中文说明
- 相同的 10 个渐进示例 + 框架实现

## 项目结构
与英文版相同，见 `ai-agents-from-scratch/01-概览.md`

## 核心价值
作为中文参考，帮助理解：
1. **Runnable 接口**: 万物可组合的统一抽象
2. **类型化消息**: System/Human/AI/Tool 四种角色
3. **提示模板**: 自动变量检测 + 格式化
4. **输出解析器**: 从 LLM 混乱输出中提取结构化数据
5. **ReAct 模式**: Thought → Action → Observation 循环
6. **记忆持久化**: JSON 文件 + 类型分类

## 教学方法
每个教程课程：
1. **lesson.md**: 为什么重要 + 解决什么问题 + 设计模式
2. **exercises/**: TODO 存根 + 内置测试
3. **solutions/**: 参考实现

渐进难度：先实现 `_call()`，然后 `pipe()` 组合，然后 `batch()` 并行，最后完整集成。
