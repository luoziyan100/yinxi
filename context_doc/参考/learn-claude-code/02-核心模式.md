## 模式 1：最小 Agent 循环 (s01)

```python
# 41 行实现一个完整 Agent
while True:
    response = LLM(messages, tools=[bash])
    if response.stop_reason == "tool_use":
        execute_bash()
        append_result_to_messages()
    else:
        break
```
**金句**: "One loop & Bash is all you need"

## 模式 4：子 Agent 上下文隔离 (s04)

```python
def run_subagent(prompt):
    sub_messages = [{"role": "user", "content": prompt}]  # 全新上下文！
    while True:
        response = LLM(messages=sub_messages)
        if tool_use: execute_tool()
        else: break
    return summary  # 只返回摘要给父级
```

```
父 Agent:              子 Agent:
messages=[...]         messages=[] (全新)
  +--> task("do X")      |
  |                      v
  |                  run_subagent()
  |                    while tool_use:
  |                       execute_tool()
  |                  return summary
  v
仅追加摘要，上下文保持干净
```
**金句**: "Break big tasks down; each subtask gets a clean context"

## 模式 5：按需技能注入 (s05)

两层注入：
1. **系统提示**: 只放技能名称 (~100 tokens/skill)
2. **tool_result**: 需要时加载完整 SKILL.md

```python
class SkillLoader:
    def load(name):
        return read("skills/{name}/SKILL.md")  # YAML frontmatter 解析
```
**金句**: "Load knowledge when you need it, not upfront"

## 模式 6：三层上下文压缩 (s06)

```
1. Micro-compact: 逐条总结最旧的轮次
2. Auto-compact: tokens > 80% 限制时，压缩所有已完成任务
3. Archival: 将旧任务移动到归档文件
```
**金句**: "Context will fill up; you need a way to make room"

## 模式 11：自治轮询 (s11)

```python
while idle_timeout < 60:
    messages = read_inbox(name)     # 检查邮箱
    if messages: resume_work(); break

    tasks = scan_task_board()       # 扫描任务板
    if unclaimed := [t for t in tasks if t.owner == ""]:
        claim(unclaimed[0]); break

    sleep(POLL_INTERVAL)            # 5 秒间隔
# 超时 60 秒 → 关闭
```
