# Learn Claude Code - 项目概览

## 基本信息
- **来源**: 教学项目
- **语言**: Python (Agent 实现) + TypeScript/Next.js (学习平台)
- **定位**: 从 0→1 拆解 Claude Code 的 Agent 架构，12 个渐进式课程
- **核心理念**: "模型就是 Agent，我们的工作是给它工具然后让开"

## 12 课渐进结构

| 阶段 | 课程 | 核心添加 |
|------|------|---------|
| **循环基础** | s01 Agent Loop | 最小内核：1 个循环 + bash |
| | s02 Tool Use | 工具分发 map |
| **规划与知识** | s03 TodoWrite | 可见进度追踪 |
| | s04 Subagents | 上下文隔离 |
| | s05 Skills | 按需知识注入 |
| | s06 Context Compact | 无限长度会话 |
| | s07 Task System | 持久化任务图 |
| **并发** | s08 Background Tasks | 非阻塞异步执行 |
| **协作** | s09 Agent Teams | 持久化队友 + 异步通信 |
| | s10 Team Protocols | 正式化通信协议 |
| | s11 Autonomous Agents | 自治发现工作 |
| | s12 Worktree Isolation | 目录隔离执行 |

## 教学理念
- 每课只添加一个机制
- 循环结构永不改变
- 工具持续累积
- 金句总结核心洞见
